# Trabajo grupal

Nuestra animación lo que hace es mostrarte como se vería el código en pantalla

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install foobar
```

## Usage

```python
import 01

# returns 'input'
01.getElementById('input')

# returns 'result'
01.getElementById('result')

01.run()
```
